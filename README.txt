Columns :E
==========

Introduction
------------
Columns :E is a C++ game that aims to mimic the gameplay of the popular game Columns.

Requires
--------
SFML v2.0 API (http://www.sfml-dev.org/download.php)

Progress
--------
- Left and Right movement of falling block
- Up and Down movement to switch order of falling blocks
- Checking for horizontal/vertical/diagnol matching blocks
